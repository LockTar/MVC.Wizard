MVC Wizard Sample Readme

This is the sample NuGet package for the MVC.Wizard NuGet package.
To use this package follow these steps below:

1. This package works out of the box with the MVC web application internet template. 
   This sample package is build on the default MVC 5 internet template in VS2013. 
   It requires the internet template because the Sample View needs a _Layout page.
2. Open the SampleWizardController and run the Sample MVC action.
